Polynomial proprioception for robot policies

PRISM exposes physical interactions from proprioception.

A lightweight polynomial conditioning module that helps sensor-minimal locomotion and vision-language-action policies reason over multiplicative structure in robot state.

Sean Lee and collaborators

Method

PRISM changes the representation of proprioception, not the sensing interface, action space, or low-level controller.

Locomotion

We add PRISM to BFM-Zero and compare against both the standard BFM-Zero model and a larger-capacity BFM-Zero control. Evaluation is aligned at 9.6M environment steps and averaged across nominal, low-friction, and payload-mass settings.

Manipulation

We add PRISM to SmolVLA by applying polynomial features only to proprioceptive state. SmolVLM features remain unchanged, while the action expert and proprioception conditioner are tuned on LIBERO multi-task demonstrations.

Stronger Backbone Validation

The original controlled experiments isolate the architectural effect. These additional experiments test whether the same polynomial bias remains useful with stronger modern backbones.
